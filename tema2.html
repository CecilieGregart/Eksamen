Tema 2 · Grundlæggende web

Løsning

I dette tema arbejdede jeg med grundlæggende web med fokus på HTML og CSS. Opgaven tog udgangspunkt i et mobildesign, som efterfølgende skulle tilpasses desktop-visning, og havde til formål at give en grundlæggende forståelse for opbygning af et responsivt website.

Websitet blev opbygget med semantisk HTML og struktureret CSS, hvor jeg arbejdede med farver, typografi og billedplacering. Projektet bestod af flere HTML-sider, som tilsammen dannede et sammenhængende site, og blev til sidst publiceret via GitHub Pages.

Proces

Processen i dette tema var tæt styret af den udleverede opgaveformulering og fungerede samtidig som min studiestartsprøve. Her skulle jeg demonstrere, at jeg havde forstået de grundlæggende principper inden for HTML og CSS.

Vi arbejdede ud fra færdige wireframes og layoutdiagrammer, som viste både mobil- og desktopversioner af sitet. Layoutdiagrammerne var opbygget med CSS Grid og Flexbox, og det var vigtigt at følge de angivne class- og id-navne for at opnå den korrekte struktur.

I arbejdet med sitet tog jeg udgangspunkt i mobile first, hvor jeg først fik mobilvisningen til at fungere og derefter udvidede layoutet til desktop ved hjælp af media queries.

Undervejs lavede jeg øvelser, hvor jeg blandt andet arbejdede med:

  • Grid-layouts, der ændrede struktur fra mobil til desktop
  • Placering af tekst og billeder i forskellige kolonneopsætninger
  • Hvordan indhold kan flyttes fra at stå under hinanden til at stå side om side

Derudover arbejdede jeg med styling af tekst, herunder line-height, letter-spacing, tekstlinks, baggrundsbilleder og webfonte. Som en del af opgaven skulle HTML-koden validere uden fejl, og sitet måtte ikke indekseres af søgemaskiner.

Læring

Tema 2 har givet mig en grundlæggende forståelse for, hvordan websites opbygges med HTML og CSS, og hvordan struktur og layout hænger sammen.

Jeg har fået praktisk erfaring med responsivt webdesign og arbejdet med CSS Grid, Flexbox og media queries, som gør det muligt at tilpasse et layout til forskellige skærmstørrelser.

Selvom opgaven var relativt styret, gav den mig et solidt fundament, som jeg har bygget videre på i de efterfølgende temaer.
