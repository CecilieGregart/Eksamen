Grundlæggende web

Løsning

I dette tema arbejdede jeg med grundlæggende web med fokus på HTML og CSS. Opgaven tog udgangspunkt i et mobildesign, som efterfølgende skulle tilpasses desktop-visning. En central del af opgaven var at sikre, at sitet fungerede responsivt og fulgte det udleverede wireframe og layoutdiagram.

Websitet blev opbygget med semantisk HTML og struktureret CSS, hvor jeg arbejdede med farver, typografi og billedplacering. Der blev anvendt to forskellige fonte, og sitet blev opdelt i flere HTML-sider, som tilsammen dannede et sammenhængende website. Afslutningsvis blev projektet uploadet via GitHub og publiceret med GitHub Pages.

Proces

Processen i dette tema var tæt styret af den udleverede opgaveformulering og fungerede samtidig som min studiestartsprøve. Formålet var at demonstrere, at jeg havde forstået de grundlæggende teknikker inden for webudvikling og kunne anvende dem korrekt.

Vi arbejdede ud fra færdige wireframes og layoutdiagrammer, som viste både mobil- og desktopversioner af sitet. Layoutdiagrammerne var opbygget med brug af CSS Grid og Flexbox, og vi skulle følge de angivne class- og id-navne for at opnå den korrekte struktur.

Undervejs lavede jeg mange øvelser, hvor jeg eksperimenterede med:

  • grid-layouts, der ændrede struktur fra mobil til desktop
  • placering af tekst og billeder i forskellige kolonneopsætninger
  • hvordan indhold kan flyttes fra at stå under hinanden til at stå side om side

Derudover arbejdede jeg med styling af tekst, herunder line-height, letter-spacing, tekstlinks, baggrundsbilleder og webfonte.

Som en del af opgaven skulle HTML-koden validere uden fejl, og sitet måtte ikke indekseres af søgemaskiner.

Link til studiestartsprøve: https://ceciliegregart.github.io/studiestartstest/

Læring

Gennem Tema 2 har jeg opnået en grundlæggende forståelse for, hvordan websites opbygges med HTML og CSS. Jeg har lært at arbejde struktureret med layoutdiagrammer, wireframes og responsive designprincipper.

Særligt arbejdet med CSS Grid og Flexbox har givet mig en bedre forståelse for, hvordan man opbygger fleksible layouts, der tilpasser sig forskellige skærmstørrelser.

Selvom opgaven var relativt styret, gav den mig et solidt fundament, som jeg har bygget videre på i de efterfølgende temaer.
